# LibPNET

A networking library written in C++

## Quick Start

1. Generate Makefile

   ```cmake
   cmake -S . -B build -SHARED=<true or false> -DCMAKE_INSTALL_PREFIX=<install_dir>
   ```

2. Build library

   ```cmake
   cmake --build build
   ```

3. Install library

   ```cmake
   cmake --install build
   ```
